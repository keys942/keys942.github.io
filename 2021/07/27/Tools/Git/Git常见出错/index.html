Git常见出错

1、连接超时

1）错误

2）原因分析

连接超时 。

3）解决办法

重试

2、SSL验证

1）错误

2）原因分析

服务器的 SSL 证书没有经过第三方机构的签署 。

3）解决办法

修改设置，解除 SSL 验证

3、远端名称不符

1）错误

2）原因分析

GitHub 远端仓库分支由原来的 master 改为了 main，push 时还使用 master 就会导致名称不符 。

3）解决办法

将 master 改为了 main

4、输入不可见字符

出现不可见字符，导致仓库路径错误 。

3）解决办法

去掉不可见字符 。
