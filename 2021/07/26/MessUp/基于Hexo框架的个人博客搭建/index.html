基于Hexo框架的个人博客搭建

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

这里，只是简单地介绍了基于 Hexo 框架的个人博客搭建，更为详细、具体的命令和操作可以去 Hexo 官网查看。

Hexo 官网：Hexo

一、依赖软件安装

Hexo 的安装和运行依赖 Node.js 和 Git 两个软件：

  • 安装官网：Node.js (nodejs.org)

  • 安装官网：Git (git-scm.com)

1、Git 的安装

1）下载相应版本，进行安装，所有设置可以默认。

2）打开 bash 或者 cmd，查看 Git 是否安装成功。

关于 Git 和 Gitee 的配置和使用可以看《Git-Gitee初步配置及使用》。

2、Node.js 的安装

1）下载 LTS 版本，进行安装，所有设置可以默认。

2）打开 bash 或者 cmd，查看 node 和 npm 是否安装成功

安装 node 的同时也安装了 npm ，npm 是 nodejs 的包管理工具。

如果 cmd 中运行以上命令显示不是内部或外部命令，也不是可运行的程序或批处理文件，那么可能安装时添加环境变量失败，可以自行添加或者在开始的最近添加中找到并打开 Node.js command prompt ，在其中使用命令行。

3）由于国外镜像源很慢，用 npm 国内淘宝的源全局安装 cnpm ，以代替 npm 。

这里也可以不安装 cnpm ，只配置 npm 国内淘宝的源，以加快下载速度。
