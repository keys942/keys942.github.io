Android studio安装

Android studio安装（Win10）

Android Studio官网：Download Android Studio and SDK tools | Android Developers (google.cn)

这里我安装的是： android-studio-2020.3.1.24-windows.exe 。

安装

双击进行安装，Next -> Next -> Next（可以修改安装目录） -> Install -> Next -> finish ，完成安装。

配置

1、选择 Do not import settings ，点击 OK 。（第一次安装没什么配置文件需要导入，所以我们选择不导入）

2、选择 Don’t sent 。 （选择不发送，因为可能连不上Google，导致后台一直在请求）

3、选择 Cancel 。（显示没找到SDK，这里先不管，后面再安装）

4、点击 Next ，选择 Custom ，点击 Next 。（这里我选择自定义，标准的话一些东西不需要配置，但是SDK可能会安装到C盘）

5、选择 jdk ，这里我选择 Android Studio 自带的，当然也可以选择自己下载的 。

6、安装 SDK ，建议全选吧，免得后面再安装麻烦，当然现在不安装后面还可以安装 。安装目录可以更改一下。

可能会出现有些选项选不上，那就不选了，不用担心，后面会提示你安装的，点击下载就好了 。

7、默认就好，Next ，等待安装完成，Finish 。

新建项目

1、选择 New Project 新建工程，选择 Empty Activity ，Next 。

2、填写相关信息，Finish 。

3、等待 Gradle 构建完成，可能会报错 ，按照提示安装丢失的 SDK 包 。

4、完成 。

添加虚拟机

这里可以通过连接手机直接进行测试，从而跳过添加虚拟机 。

1、找到下图倒数第二个 AVD Manager，单击打开

2、单击 Create Virtual Devices 添加虚拟机 。

3、选择机型和系统内核（系统内核需要下载，选择想用的点 Download 下载就可）

运行

点击小锤子进行创建工程，点击三角形进行运行，得到运行在虚拟手机上的效果 。
