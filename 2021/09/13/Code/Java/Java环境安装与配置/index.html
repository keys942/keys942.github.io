安装 JDK 并配置 Java 环境

可以自行百度搜索，下载相应版本的 JDK ，这里我安装的 jdk-8u301-windows-x64.exe 。

Java 8 下载地址：Java SE Development Kit 8 - Downloads (oracle.com)

Java 下载地址：Java SE - Downloads | Oracle Technology Network | Oracle

安装

这里建议直接默认安装，如果 C 盘 空间不够，可以更换其他目录，但一定要记住安装路径，默认安装一般会安装在 C:\Program Files\Java 。

最好安装在 C 盘，以免后面出现问题 。
